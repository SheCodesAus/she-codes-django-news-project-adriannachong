To activate virtualenv in terminal:
    source venv/bin/activate

Default django project page:
    http://localhost:8000/ 

When making changes to any '.py' file must make migrations
    python3 manage.py makemigrations
    python3 manage.py migrate

Remember to run server when making changes and following the steps
    python3 manage.py runserver
